# cmd

## ¿Qué es cmd?

**cmd** es un comando directo de i-HakLab que permite gestionar ajustes principales de Android directamente desde Termux. Proporciona acceso a configuraciones del sistema Android a través de la línea de comandos.

## ¿Para qué es útil?

- Gestionar configuraciones de Android sin salir de Termux
- Acceder a ajustes del sistema desde scripts

## ¿Cómo se usa?

```bash
cmd
```

Ejecuta el comando y sigue las opciones disponibles en pantalla.
