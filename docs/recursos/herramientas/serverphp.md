# serverphp

## ¿Qué es serverphp?

**serverphp** es un comando directo de i-HakLab que inicia el servidor PHP integrado, permitiendo ejecutar aplicaciones PHP sin necesidad de configurar Apache.

## ¿Para qué es útil?

- Iniciar el servidor de desarrollo PHP
- Probar aplicaciones PHP localmente
- Servir páginas PHP para laboratorios de prueba

## ¿Cómo se usa?

```bash
serverphp
```

---

*Nota: Comando directo del ecosistema i-HakLab.*
