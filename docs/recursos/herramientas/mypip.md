# mypip

## ¿Qué es mypip?

**mypip** es un comando directo de i-HakLab que muestra la dirección IP pública del dispositivo consultando servicios externos.

## ¿Para qué es útil?

- Conocer la IP pública para configurar servicios externos
- Verificar la conexión a Internet
- Diagnosticar problemas de red

## ¿Cómo se usa?

```bash
mypip
```
