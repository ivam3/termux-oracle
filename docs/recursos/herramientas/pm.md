# pm

## ¿Qué es pm?

**pm** (Package Manager) es un comando directo de i-HakLab que proporciona acceso al gestor de paquetes de Android desde Termux, permitiendo listar, instalar y desinstalar aplicaciones del sistema Android.

## ¿Para qué es útil?

- Listar paquetes de Android instalados
- Desinstalar aplicaciones del sistema
- Gestionar permisos de aplicaciones

## ¿Cómo se usa?

```bash
pm list packages
pm uninstall --user 0 <package>
```
