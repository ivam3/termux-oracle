# du

## ¿Qué es du?

**du** es un comando directo de i-HakLab que muestra el peso de directorios y archivos de forma recursiva. En i-HakLab está configurado como `du -hP` para mostrar tamaños en formato legible.

## ¿Para qué es útil?

- Calcular el espacio ocupado por directorios
- Identificar archivos y carpetas que consumen más almacenamiento

## ¿Cómo se usa?

```bash
du
du ~/storage
du -hP ~/projects --max-depth=2
```

---

*Nota: Comando directo del ecosistema i-HakLab.*
