# fixer

## ¿Qué es fixer?

**fixer** es un comando directo de i-HakLab que automatiza la solución de problemas comunes de Termux, incluyendo reparación de repositorios, limpieza de caché y restauración de configuraciones dañadas.

## ¿Para qué es útil?

- Reparar repositorios APT dañados
- Solucionar errores de dependencias rotas
- Restaurar configuraciones por defecto de Termux
- Limpiar caché y archivos temporales del sistema

## ¿Cómo se usa?

```bash
fixer
```

Sigue las instrucciones interactivas para seleccionar el tipo de reparación.
