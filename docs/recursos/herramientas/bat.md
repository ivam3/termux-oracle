# bat

## ¿Qué es bat?

**bat** es un comando directo de i-HakLab que reemplaza a `cat` con resaltado de sintaxis, numeración de líneas y soporte para temas de colores. En i-HakLab está configurado con el tema `Visual Studio Dark+`.

## ¿Para qué es útil?

- Ver archivos con resaltado de sintaxis automático
- Mostrar números de línea para depuración
- Leer múltiples archivos con paginación integrada

## ¿Cómo se usa?

```bash
bat archivo.txt
cat README.md | bat
```
