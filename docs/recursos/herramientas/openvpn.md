# openvpn

## ¿Qué es openvpn?

**openvpn** es un comando directo de i-HakLab que abre la aplicación Android de OpenVPN, facilitando la conexión a redes VPN desde Termux.

## ¿Para qué es útil?

- Iniciar la app OpenVPN desde la terminal
- Integrar conexiones VPN en flujos de trabajo

## ¿Cómo se usa?

```bash
openvpn
```
