# LOCALHOST

## ¿Qué es LOCALHOST?

**LOCALHOST** es un comando directo de i-HakLab que muestra la dirección IP actual del dispositivo en la red LAN.

## ¿Para qué es útil?

- Conocer la IP local para servicios en red
- Configurar servidores y pruebas de conexión local
- Compartir recursos en la red local

## ¿Cómo se usa?

```bash
LOCALHOST
```

---

*Nota: Comando directo del ecosistema i-HakLab.*
