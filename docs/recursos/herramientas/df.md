# df

## ¿Qué es df?

**df** es un comando directo de i-HakLab que muestra información sobre la memoria montada y el uso del espacio en disco. En i-HakLab está configurado como alias de `duf` para una salida más amigable.

## ¿Para qué es útil?

- Ver el espacio disponible en dispositivos de almacenamiento
- Monitorear el uso del disco en Termux y Android

## ¿Cómo se usa?

```bash
df
```
