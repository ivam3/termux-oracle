# phantom-ps

## ¿Qué es phantom-ps?

**phantom-ps** es un comando directo de i-HakLab que cambia el límite de procesos fantasma del sistema a `2147483647`, permitiendo un mayor número de procesos en segundo plano en Termux.

## ¿Para qué es útil?

- Aumentar el límite de procesos disponibles
- Evitar errores de "out of memory" o límite de procesos
- Mejorar el rendimiento en tareas que requieren múltiples procesos

## Consideraciones

Requiere pantalla dividida o acceso a opciones de desarrollador de Android para funcionar correctamente.

```bash
phantom-ps
```
