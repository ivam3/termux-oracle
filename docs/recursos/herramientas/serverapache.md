# serverapache

## ¿Qué es serverapache?

**serverapache** es un comando directo de i-HakLab que inicia el servidor web Apache, permitiendo servir páginas web y archivos desde Termux.

## ¿Para qué es útil?

- Iniciar Apache para pruebas locales
- Servir aplicaciones web en desarrollo
- Compartir archivos vía HTTP en la red local

## ¿Cómo se usa?

```bash
serverapache
```
