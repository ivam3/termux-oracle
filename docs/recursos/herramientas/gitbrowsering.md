# gitbrowsering

## ¿Qué es gitbrowsering?

**gitbrowsering** es un comando directo de i-HakLab que permite buscar repositorios en GitHub directamente desde la terminal, sin necesidad de abrir un navegador.

## ¿Para qué es útil?

- Buscar repositorios por nombre o tema
- Explorar proyectos de GitHub desde la CLI
- Encontrar herramientas y frameworks rápidamente

## ¿Cómo se usa?

```bash
gitbrowsering
```

Ingresa el término de búsqueda cuando se solicite.
