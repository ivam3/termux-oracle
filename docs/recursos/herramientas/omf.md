# omf

## ¿Qué es omf?

**omf** (Oh My Fish) es un comando directo de i-HakLab que permite cambiar el tema de Oh My Fish, el framework de personalización para la shell Fish que es la shell predeterminada de i-HakLab.

## ¿Para qué es útil?

- Cambiar la apariencia de la terminal
- Personalizar el prompt de Fish con temas predefinidos

## ¿Cómo se usa?

```bash
omf theme
omf install <nombre_tema>
```
