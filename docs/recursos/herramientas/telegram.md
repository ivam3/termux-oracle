# telegram

## ¿Qué es telegram?

**telegram** es un comando directo de i-HakLab que abre la aplicación Android de Telegram desde la terminal.

## ¿Para qué es útil?

- Abrir Telegram rápidamente desde Termux
- Integrar acceso a mensajería en flujos de trabajo

## ¿Cómo se usa?

```bash
telegram
```
