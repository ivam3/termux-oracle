# adminfiles

## ¿Qué es adminfiles?

**adminfiles** es un comando directo de i-HakLab que copia archivos desde el almacenamiento interno o externo de Android al directorio de Termux, facilitando la transferencia de datos sin necesidad de comandos complejos.

## ¿Para qué es útil?

- Copiar archivos del almacenamiento interno/externo a Termux
- Transferir scripts, capturas, descargas al entorno de trabajo

## ¿Cómo se usa?

```bash
adminfiles
```

Sigue las instrucciones interactivas para seleccionar el origen y destino.
