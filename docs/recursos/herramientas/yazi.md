# yazi

## ¿Qué es yazi?

**yazi** es un gestor de archivos en terminal de alto rendimiento escrito en Rust. i-HakLab lo incluye como comando directo para navegar y gestionar archivos de forma visual desde la terminal.

## ¿Para qué es útil?

- Navegar por el sistema de archivos con interfaz TUI
- Previsualizar archivos (imágenes, texto, PDFs)
- Copiar, mover, renombrar y eliminar archivos
- Integrar con visores y editores externos

## ¿Cómo se usa?

```bash
yazi
yazi ~/projects
```
