# rmcache

## ¿Qué es rmcache?

**rmcache** es un comando directo de i-HakLab que elimina la caché de la sesión actual, archivos temporales y paquetes residuales de APT, liberando espacio de almacenamiento.

## ¿Para qué es útil?

- Liberar espacio en disco eliminando cachés
- Limpiar paquetes APT descargados pero no necesarios
- Mantener el sistema optimizado

## ¿Cómo se usa?

```bash
rmcache
```

---

*Nota: Comando directo del ecosistema i-HakLab.*
