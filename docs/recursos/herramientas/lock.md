# lock

## ¿Qué es lock?

**lock** es un comando directo de i-HakLab que bloquea la pantalla de Termux, requiriendo la contraseña configurada con `i-HakLab passwd set` para desbloquear.

## ¿Para qué es útil?

- Bloquear la sesión de Termux al alejarse del dispositivo
- Proteger el acceso a la terminal con contraseña o huella

## ¿Cómo se usa?

```bash
lock
```

---

*Nota: Comando directo del ecosistema i-HakLab.*
